Thomas Sewell's Portfolio

Hello! My name is Thomas Jacob Sewell. I am an aspiring Game developer who is currently enrolled at NCTC. I can comfortably use both Unity and Unreal to create games using their various systems. In fact I have already made some games before.

Prism Platform

Prism Platform is a game where you navigate a series of puzzels by affecting the layout by swapping the color of the background revealing aspects otherwise hidden.

Color Blaster

Color Blaster is a first person shooter in which the player is challenged to shoot as many colored targets as possible with the same color in order to earn as many points as possible before the time runs out.

Potion Mishaps

Potion Mishaps is a potion making game in which you take a few ingredients and go through a couple steps in order to make potions of a well sized variety.
